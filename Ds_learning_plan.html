How to use this plan

  1. One notebook per session. Open Jupyter or VS Code, create a new .ipynb, paste the AI prompt at the bottom of the card, and work through the generated notebook.
  2. Don't just run cells — edit them. Change numbers, break things, fix them. The learning is in the friction.
